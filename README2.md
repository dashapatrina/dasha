
# Лабораторная работа 1.2

### Мультисписок

Мультисписок — Многомерная структура данных, представляющая собой вложенную коллекцию элементов (списки внутри списков). Элементы могут иметь произвольную глубину вложения.

1.Python : Необходимо создать структуру, представленную в виде вложенного списка (multi-dimensional array), а также научиться обращаться к каждому элементу отдельно.

2.Java : Java поддерживает создание вложенных коллекций через List.

3.C++ : В C++ такая структура реализована через контейнеры типа vector или list.

Очередь — Линейная структура данных, организованная по принципу FIFO ("First In First Out"). Добавление элементов происходит в конец очереди, а удаление — из её начала.

1.Python : Реализуем простую очередь с использованием встроенной библиотеки collections.deque.

2.Java : Класс LinkedList удобно подходит для реализации очередей.

3.C++ : Использование стандартной библиотеки <queue> для организации очереди.

Дек (двусторонняя очередь) — Двухсторонняя очередь, позволяющая вставлять и удалять элементы как с начала, так и с конца очереди.

1.Python : Используя тот же deque, реализуем возможности добавления и удаления элементов с обоих концов.

2.Java : Класс ArrayDeque отлично справляется с задачей поддержки обеих сторон очереди.

3.C++ : Библиотека <deque> реализует двухстороннюю очередь.

Приоритетная очередь — Структура данных, хранящая элементы согласно их приоритету. Элемент с наивысшим приоритетом доступен первым. Обычно реализуется с помощью двоичной кучи (heap).

1.Python : Python предоставляет модуль queue для создания приоритетных очередей (PriorityQueue), где элементы сортируются по указанному приоритету.

2.Java : Стандартная библиотека Java включает класс PriorityQueue, поддерживающий сортировку по приоритету.

3.C++ : Приоритетная очередь доступна через стандартную библиотеку <queue>. Здесь применяется max-heap по умолчанию.
